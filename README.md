# Example_Todo-List
This project focuses on learning how to create todo-list with Vue framework, covering basic design with Daisy UI.

## Tech Stack

✔️ HTML CSS JS 
✔️ Vue with Pinia
✔️ Tailwind and Daisy UI
✔️ Mock API  

## Deploy URL
  https://sample-todo-list.vercel.app/
